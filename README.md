# Django Learning Journey

This repository is my personal space for documenting and practicing everything I learn while working with Django. It will grow over time as I build projects, solve problems, and improve my understanding of the framework.

## What this journey is about

I am learning Django step by step, starting from the basics and moving toward more advanced topics such as:

- project structure and apps
- models, views, templates, and URLs
- database design and migrations
- forms and validation
- authentication and authorization
- testing and deployment
- building practical web applications

## Goals

My main goals for this journey are to:

- understand how Django works behind the scenes
- build small but meaningful projects
- strengthen my backend development skills
- learn clean and maintainable coding habits
- create a record of my progress over time

## Learning roadmap

1. Learn the fundamentals of Django
2. Build simple applications and practice CRUD
3. Work with databases and migrations
4. Add user authentication and forms
5. Explore testing, deployment, and real-world best practices

## Current focus

At the moment, I am focusing on understanding the core Django workflow and getting comfortable with the MVC-style structure that Django uses.

## How I will use this repository

This repository will serve as:

- a learning log
- a place for practice projects
- a collection of useful code examples
- a record of milestones and progress

## Getting started

When new Django projects are added, they will be organized in a way that makes it easy to follow the progression of the learning journey.

Typical setup for a new Django project:

```bash
python -m venv .venv
source .venv/bin/activate
pip install django
```

On Windows, activate the environment with:

```bash
.venv\Scripts\activate
```
